# PRODIGY_SD_02
This project was completed as part of the Prodigy InfoTech Software Development Internship.
## Task Description
This Python program generates a random number and asks the user to guess it. It provides feedback if the guess is too high or too low until the correct number is guessed.
## Features
- Random number generation
- User input for guesses
- "Too High" and "Too Low" hints
- Counts the number of attempts
- Simple command-line interface
## Technologies Used
- Python 3
## How to Run
1. Clone this repository.
2. Open the project folder.
3. Run:
```bash
python guessing_game.py
```
4. Enter your guesses until you find the correct number.
